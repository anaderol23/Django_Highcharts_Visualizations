AWS Honeypot Attacks Dashboard

Overview

This project contains a few HighCharts visualizations based on the "AWS Honeypot Attacks" dataset from Kaggle.

Front-End: HTML, JavaScript, Bootstrap, jQuery

Back-End: Python, Django

Content

The AWS Honeypot Database is an open-source database including information on cyber attacks/attempts.

The data consists in 451,581 data points collected from 9:53pm on 3 March 2013 to 5:55am on 8 September 2013.

However, since this project's purpose is to demo my freshly developed web development skills, I have only loaded 54329 data points.

Honeypot attacks definition

As defined on Wikipedia, a "honeypot attack" is a "computer security mechanism set to detect, deflect, or, in some manner, counteract attempts at unauthorized use of information systems. Generally, a honeypot consists of data (for example, in a network site) that appears to be a legitimate part of the site that seems to contain information or a resource of value to attackers, but actually, is isolated and monitored and, enables blocking or analyzing the attackers. This is similar to police sting operations, colloquially known as "baiting" a suspect."
